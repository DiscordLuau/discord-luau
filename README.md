DiscordLuau is a Discord bot library written in Luau, designed for use with [Zune](https://zune.dev).

## Getting Started

Everything you need - installation, your first bot, guides for messages, embeds, attachments, and the full API reference - is in the documentation:

**[discordluau-docs.devcomp.workers.dev](https://discordluau-docs.devcomp.workers.dev/)**

Start with the [Installation](https://discordluau-docs.devcomp.workers.dev/getting-started/installation/) page, then follow the [Creating a Bot](https://discordluau-docs.devcomp.workers.dev/guides/creating-a-bot/) guide.
